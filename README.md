# 课程简介

SQLite数据库加密了，怎么办？

没关系，不需要密码，也不需要解密，让我们直接读取里面的内容吧！

掌握这个技能，只要它是SQLite数据库，只要它能被“专有”的程序访问，那么，无论它采取多么“强”的加密方式，在你面前，它就是一个普通数据库而已，里面的数据可由你任你摆布！
